Step-by-Step Usage

For Desktop

  1. Copy or download the Python script above.
  2. Transfer to the target computer (USB, email, etc).
  3. Open terminal/command prompt and run:
    python crypto_desktop_scan.py
  4. The scan report is saved as crypto_forensics_report.txt. Open it to review all traces and evidence.

For Android

  1. Enable Developer Mode & USB Debugging.
  2. Connect device to PC and use adb, or open Termux app.
  3. Copy or download the shell script above.
  4. In adb:
    adb push crypto_android_scan.sh /data/local/tmp/
    adb shell sh /data/local/tmp/crypto_android_scan.sh > /sdcard/crypto_android_report.txt
  5. In Termux:
    sh crypto_android_scan.sh > crypto_android_report.txt
  6. Review crypto_android_report.txt for all detected traces.
