Desktop Forensics Script Win/Linux/macOS

Detects crypto wallets, browser extensions, config files, screenshots, clipboard, notes, browser history, password managers, and more. Requires Python 3.7+

Scans for:

  • Installed wallet apps & password managers
  • Wallet/seed/config files (with crypto content check)
  • Clipboard, screenshots, browser history, notes
  • SHA256 hash as evidence for each crypto-related file

Android Forensics Script ADB/Termux

Detects wallet apps, APKs, wallet files, screenshots, WhatsApp, SMS traces, and clipboard artifacts. Requires adb or Termux

Scans for:

  • All crypto wallet apps (installed & recently uninstalled)
  • Wallet/seed/APKs/files/screenshots (with hashes)
  • WhatsApp/Downloads/Documents/SMS
  • Clipboard artifacts (Termux-API)
  • SHA256 for evidence chain

Step-by-Step Usage

For Desktop

  1. Click "Download Python" if not already installed (add to PATH during setup).
  2. Copy or download the Python script above.
  3. Transfer to the target computer (USB, email, etc).
  4. Open terminal/command prompt and run:
    python crypto_desktop_scan.py
  5. The scan report is saved as crypto_forensics_report.txt. Open it to review all traces and evidence.

For Android

  1. Enable Developer Mode & USB Debugging.
  2. Connect device to PC and use adb, or open Termux app.
  3. Copy or download the shell script above.
  4. In adb:
    adb push crypto_android_scan.sh /data/local/tmp/
    adb shell sh /data/local/tmp/crypto_android_scan.sh > /sdcard/crypto_android_report.txt
  5. In Termux:
    sh crypto_android_scan.sh > crypto_android_report.txt
  6. Review crypto_android_report.txt for all detected traces.
