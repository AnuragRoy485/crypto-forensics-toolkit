Android Forensics Script ADB/Termux

Detects wallet apps, wallet files, screenshots, WhatsApp, SMS traces, clipboard artifacts. Works via ADB or Termux

How does this work?
  • Finds wallet apps, APK traces, and files with crypto keywords
  • Hashes detected files for integrity/evidence
  • Reports results to text output

Step-by-Step Usage Guide

  1. Desktop Scan:
    • Download Python 3.7+ () if not already installed.
    • Click "Download Script" under Desktop and save crypto_desktop_scan.py.
    • Open PowerShell/Terminal, go to Downloads folder, run:
      python crypto_desktop_scan.py
    • Review crypto_forensics_report.txt for detected crypto evidence.
    • Paste contents of this report above to get a forensic summary.
  2. Android Scan:
    • Enable Developer Mode & USB Debugging on the device.
    • Connect device to PC, use ADB or Termux app.
    • Download and push the shell script, run:
      sh crypto_android_scan.sh > crypto_android_report.txt
    • Review crypto_android_report.txt for evidence.
